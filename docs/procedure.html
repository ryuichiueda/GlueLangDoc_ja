5. 手続き¶

　手続きはシェルの「関数」に相当するものです。 次のように定義、実行します。 実行するときは、このスクリプトを指す this という接頭語を付けます。

  • Fig.: proc.glue
#definition
proc f = /bin/echo 'HELLO'

#execution
this.f

実行すると手続きの中のechoが呼び出されます。

$ glue proc1.glue
HELLO

　長いものはdoをつけて改行し、インデントを合わせて記述します。

proc hoge = do
  /bin/echo 'abc' >>= /usr/bin/rev

このスクリプトの出力は次のようになります。

$ glue proc2.glue
